# IBM Information Technology - Core Skills Labs

This repository documents my hands-on practice, exercises, and notes while completing the **IBM Information Technology** course on SkillsBuild.  

## Course Overview
The course builds core skills required for a rewarding IT role. Among other things, it covers:  
- Provisioning, installing, and configuring hardware and software systems  
- Maintaining devices and computer networks  
- Investigating and solving technical and systematic issues  
- Supporting users through issues and request resolution  

## Objective of this Repository
- Track my learning progress through practical labs and theory notes  
- Provide proof of hands-on skills aligned with IBM course objectives  
- Showcase structured work to potential employers  

## Credential
Upon completion, I will earn a verified **IBM Information Technology** digital credential.  
The credential can be shared on LinkedIn and my resume as proof of subject matter expertise.
